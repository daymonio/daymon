Workers — give your tasks a personality

Workers are named agent profiles with system prompts. Pick a template, edit it, or write your own from scratch. Assign to any task.

Chief of Staff
Proactive business ops. Triages ruthlessly, anticipates needs, proposes actions instead of asking what to do.
Researcher
Deep research with strong opinions on sources. Synthesis over summarization — every finding gets a "so what?"
Email Assistant
3-tier triage: ACT NOW, ACT TODAY, SKIP. Drafts replies matching sender's tone. Never sends anything.
Competitor Tracker
Reports signal, not noise. Silence is a valid report. Each finding includes what changed, why it matters, and how to respond.
Code Reviewer
Catches what linters miss. Leads with blockers, cites specific lines, suggests concrete fixes. No bikeshedding.
+ Create your own
4 more built-in templates (Writer, Tech Trend Analyst, DevOps, Data Analyst) — or write any system prompt from scratch. Edit anytime.
How to use workers
Create
Pick a template or write your own system prompt — in the Daymon UI or via Claude chat.
Assign
Attach a worker to any task. "Schedule this with my Researcher worker."
Run
The worker's system prompt is injected into every execution. The task runs with that personality.
