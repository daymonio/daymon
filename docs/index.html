Daymon is a free, open-source macOS desktop app that adds scheduling, persistent memory, and background automation to Claude. It sits in your menu bar, connects to Claude Desktop or Claude Code, and runs tasks on a schedule — even when the app is closed. No API keys, no cloud servers, no account required.

100% local. No cloud. No account. No tracking.
Your data is a SQLite file on your Mac — inspect or delete it anytime.
